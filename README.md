# Flask Blog

学习《Flask Web开发实战》

参考书中Demo: http://bluelog.helloflask.com

## 运行

创建虚拟环境并导入依赖:
```
virtualenv venv
Linux: source venv/bin/activate
Windows: venv\Scripts\activate
pip install -r requirements.txt
```

运行:
```
$ flask forge
$ flask run
* Running on http://127.0.0.1:5000/
```
